# ADAAPT-Project: Trendly
This application is a way for a user to make and update their bucket list goals.

## Tech/Framework used
1. Handlebars
2. Materialize
3. Dbeaver
4. MAMP


## How to work it

1. Make Profile
2. Make Goals
3. Update Goals

### Types of goals

- []Concerts
- []Eatery
- []Locations
- []Movies
- []Books
- []Extreme Activities
- []Life
